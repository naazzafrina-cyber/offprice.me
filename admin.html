📋 How to Publish Admin Deals to GitHub Pages

  • Click "Export deals.js" below to download your deals data file.
  • Go to your GitHub repository at github.com/yourusername/offprice
  • Drag & drop the downloaded deals-data.js file into the repo root, commit it.
  • Your index.html already loads this file — all visitors will see your admin deals instantly.
  • Repeat whenever you update deals. The file replaces the old one automatically on GitHub.

🤖 Option 1 — GitHub Actions Auto-Update (Free)

Create a Python scraper that pulls deals from affiliate APIs and auto-commits the deals-data.js to your GitHub repo every Monday morning.

# .github/workflows/update-deals.yml name: Weekly Deal Update on: schedule: - cron: '0 5 * * 1' # Every Monday 5AM UTC (9AM GST) workflow_dispatch: # Also allows manual trigger jobs: update: runs-on: ubuntu-latest steps: - uses: actions/checkout@v3 - uses: actions/setup-python@v4 with: {python-version: '3.11'} - run: pip install requests admitad-api - run: python scripts/fetch_deals.py - run: | git config user.email "bot@offprice.me" git config user.name "OffPrice Bot" git add deals-data.js git diff --staged --quiet || git commit -m "Auto: Weekly deal refresh $(date +%Y-%m-%d)" git push

🏦 How to Receive Money to UAE Bank Account

  • Google AdSense → Supports direct wire transfer to UAE banks (ADCB, Emirates NBD, Mashreq, FAB, etc.). Set up in AdSense → Payments → Payment method → Add bank account. Minimum payout: $100 USD. Paid monthly.
  • Admitad.ae → Pays via PayPal, wire transfer, or Payoneer. Open a Payoneer account (free), link it to Admitad, then withdraw to your UAE bank. Payoneer → UAE bank transfer is free above $1,000.
  • Amazon Associates UAE/KSA → Pays via direct bank transfer or Amazon gift cards. UAE bank account accepted. Minimum: AED 200 or SAR 100.
  • Booking.com → Pays monthly via bank transfer in USD. Your UAE bank account (held in USD or AED) receives it. Use SWIFT/IBAN of your UAE bank.
  • Payoneer → Recommended as a universal withdrawal layer. Receive from any affiliate program globally, then transfer to Emirates NBD / ADCB / FAB — typically 1–3 business days, minimal fees.
