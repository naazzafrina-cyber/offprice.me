📋 How to Publish Admin Deals to GitHub Pages

  • Click "Download deals-data.js" below to export your deals data file.
  • Go to your GitHub repository (github.com) and open the repo root folder.
  • Drag & drop the downloaded deals-data.js into the repo, then commit.
  • The deals page automatically loads this file — visitors see your deals instantly.
  • Repeat whenever you update deals. The file replaces the old one on each commit.
